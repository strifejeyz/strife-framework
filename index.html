Strife Framework

A Fast and Lightweight PHP MVC Framework.

Smart, Handy & Neat URLs

<?php

// Using Controller
get('user-list -> /orders/:int/:any', 'OrderController@index');

// Using Closure
get('foo-bar -> /foo/bar/:str', function ($baz) {
    echo $baz;
});

// HTTP requests
post('foo-bar -> /foo/bar', function () {
    dump($_POST);
});

Query Builder & ORM

<?php

use App\Models\Order;

class OrderController {

    public function($id, $date){
        $order = Order::where('id', $id)
            ->where(order_date, $date)
            ->order('order_date', 'desc')
            ->limit(10)
            ->get();

        render('list-order', compact('order'));
    }
}

Form Builder / Interpolation / Auto-validation

The 3 essential functionality this framework have works together to prevent CSRF/XSS/Sesion Hijacking. Also provides a super-easy way to validate form fields by the use of HTTP request handler.

Form Builder/Binding
Form Builder uses Auto-interpolation of values using form binding.
Form Security
Form Builder forms are secured, making sure the person who sent it is the one who will receive it.
Auto-validation
With the help of HTTP Request Handlers, you can create Form Rules.
